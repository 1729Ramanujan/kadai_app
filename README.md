# ①課題名
時間割・課題登録アプリ（プロトタイプ）

## ②課題内容（どんな作品か）
-時間割を登録してその授業の課題を授業ごとに登録できるようにしました。

## ③工夫した点・こだわった点
- 工夫した点：
-元々Firebaseをデータベースとして作っていたものがあったので、それをPHPに書き換えました
-htmlでの表示の仕方、表の作成方法などはAIに聞きながら、一番再現性があるものを選びました


## ④難しかった点・次回トライしたいこと（又は機能）
- ここまでだとただの時間割登録アプリなので、ここからは課題にファイル追加や内容の記述をできるようにして、それをAIが自動であらかじめといておいてくれるような機能をつけたいです
-さらにUIがAIに作ってもらったハリボテなので、修正してより使いやすいようなデザインにしたいです。
